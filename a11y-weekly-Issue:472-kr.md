# 제472호

제472호는 2025년 11월 3일에 발송되었습니다.

## 주요 기사: 디자인과 개발 사이에서 접근성이 무너지는 이유

"제품이 QA에 도달할 무렵, Figma에서는 접근 가능해 보였던 기능들이 갑자기 실제 테스트에서 실패합니다. 포커스 순서가 예측 불가능하게 동작합니다. 스크린 리더가 전체 섹션을 놓칩니다. 캡션이 UI 요소와 겹칩니다. 개발자들은 한때 '이미 접근성을 고려해 디자인되었던' 것을 수정하느라 허둥대게 됩니다."

[디자인과 개발 사이에서 접근성이 무너지는 이유](https://www.smashingmagazine.com/2025/11/accessibility-breaks-between-design-development/) 더 읽기

## 스폰서: 접근성 문제가 실시간으로 해결되는 과정을 시청하세요

11월 5일 수요일 오후 1시(PST)에 라이브스트림에 참여하세요. 창립자 Natalie가 접근성 문제를 실시간으로 해결합니다. 사이트의 접근성 문제를 해결하면서 문제 해결, 코드 수정, 문제 해결 과정을 보실 수 있습니다. Natalie가 수정 프로세스를 진행하고 수정이 간단하지 않을 때 발생하는 예상치 못한 상황을 어떻게 처리하는지 지켜보세요. 실시간으로 참여할 수 없다면 녹화본을 시청하실 수 있습니다.

[YouTube에서 라이브스트림 시청하기](https://www.youtube.com/watch?v=example).
